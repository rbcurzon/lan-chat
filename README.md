# LAN Chat on Terminal
A Java application that explores network communication using the User Datagram Protocol (UDP).

## Usage

```bash
# add excutable permission
chmod +x run-server.sh
chmod +x run-chat-client.sh

# run server
./run-server.sh

# run chat-client
./run-chat-client.sh
```
